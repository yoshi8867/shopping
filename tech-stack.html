NestJS

v11
메인 프레임워크
Node.js 위에서 동작하는 TypeScript 기반 서버 프레임워크. 모듈/컨트롤러/서비스 구조로 관심사 분리.
TypeScript DI Decorator Module
🗄️

TypeORM

v0.3
ORM (Object Relational Mapper)
엔티티 클래스로 DB 테이블 정의. TypeScript 데코레이터로 관계·제약 선언. synchronize: true로 개발 중 자동 스키마 생성.
Entity Repository Relations
🟢

SQLite (better-sqlite3)

파일 DB
데이터베이스
별도 DB 서버 설치 없이 .sqlite 파일 하나로 동작. 연습 프로젝트에 최적. 실제 배포 시 PostgreSQL로 교체 가능.
파일 기반 Zero Config
🔑

JWT + Passport

@nestjs/jwt
인증 · 인가
로그인 시 JWT 발급, 이후 요청마다 Bearer 토큰으로 인증. JwtAuthGuard와 RolesGuard로 엔드포인트별 접근 제어.
JWT Bearer Token Role Guard
🔒

bcryptjs

v2
비밀번호 해싱
회원가입 시 비밀번호를 bcrypt salt=10으로 해시하여 저장. 평문 비밀번호는 DB에 절대 저장하지 않음.
Hash Salt
📋

Swagger (OpenAPI)

@nestjs/swagger
API 문서 자동 생성
컨트롤러·DTO 데코레이터에서 자동으로 API 문서 생성. /api/docs에서 브라우저로 API 직접 테스트 가능.
OpenAPI 3.0 Swagger UI
✅

class-validator

+ class-transformer
DTO 입력값 검증
DTO 클래스에 @IsEmail(), @MinLength() 등의 데코레이터로 요청 바디 자동 검증. 잘못된 입력은 400으로 자동 거절.
Validation Transform DTO
